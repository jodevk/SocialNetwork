# Simple Social Network

REST API социальной сети: регистрация, посты, подписки, лайки. JWT-аутентификация.

## Стек

- **Java 21**, **Spring Boot 3.2**
- **Spring Security** + JWT
- **Spring Data JPA**, **H2** (локально) / **PostgreSQL** (продакшен)
- **SpringDoc OpenAPI** (Swagger UI)
- **Lombok**, **Validation**
